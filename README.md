## Project Overview
Predicting student exam scores using Machine Learning and NoSQL database management.

## Objectives
- Perform comprehensive EDA on exam score dataset
- Build regression model (Linear Regression)
- Implement MongoDB for large-scale data storage
- Evaluate models using RMSE, MAE, R²

## Technologies
- Python 3.x
- Pandas, NumPy, Scikit-learn
- MongoDB Atlas
- Matplotlib, Seaborn

## Results
- MAE  : 8.70
- MSE  : 117.06
- RMSE : 10.82
- R²   : 0.6708
